== Description ==

Protect your WordPress site from spam machines by enable google recaptcha.

Simple and lightweight to install.

Free and fast.


== Installation ==

1. Upload `recaptcha.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
